Block Elements

Block elements are those elements which starts from new line,once the code is written. div tag comes under block tag.once we write the block elements then even if we wrote only one word then also it will take whole line.

Examples of Block elements

  • p
  • h1
  • h2
  • h3
Inline Elements

Inline elements are those which starts from the same line after the completion of previous element.it does not take whole space.

Examples of Inline Elements

  • strong
  • em
  • mar
  • span
